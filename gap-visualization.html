Implementation Details

  • All gaps are exactly 20px: Both horizontal and vertical gaps between all elements
  • First row: Left column has 1:1 aspect ratio (square), right column has two 2:1 landscape images stacked vertically
  • Second row: Left column has two 2:1 landscape images stacked vertically, right column has 1:1 aspect ratio (square)
  • Implementation approach:
    • CSS Grid for main layout with 20px gap between columns
    • Flexbox with 20px gap for vertical stacking within columns
    • Aspect ratios maintained with aspect-ratio property
  • Responsive: Maintains consistent 20px gaps on all screen sizes
